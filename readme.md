# OPSUCHT Utilities
Adds some ingame utilities for the OPSUCHT.net minecraft network | Not affiliated with Interwebmedia GmbH

### ✨ Features
- [x] Context menu bullets
- [x] Clickable nicknames in chat
- [x] More detailed Discord rich presence
- [x] AutoFly on join
- [x] Automatic coloring of @pings in chat
- [x] OPSUCHT Market, auction and merchant Widgets
- [ ] Auto reward claimer
- [x] Command cooldown reminders (e.g. for `/skull`)

### 📦 Installation
1. Press `Win` + `R`
2. Paste this into the window that popped up: `%appdata%/.minecraft/LabyMod-neo/addons` and press enter
3. It should open your LabyMod addon directory; Paste the [OPSUCHT-Utilities.jar](https://github.com/RappyLabyAddons/OPSUCHT-Utilities/releases/latest/download/OPSUCHT-Utilities.jar) in there.
4. Launch your LabyMod client.
